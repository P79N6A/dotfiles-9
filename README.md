dotfiles
========

## Install

Clone to local machine.

    $ git clone git://github.com/spockwangs/dotfiles.git

Copy to home directory.

    $ cd dotfiles
    $ ./install.sh

## Make local customizations

Put your customizations in dotfiles appended with `.local`:

* `~/.bashrc.local`
* `~/.gitconfig.local`
* `~/.tmux.conf.local`
* `~/.emacs.d/emacs.local`
